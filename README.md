# Taller CRUD de Usuarios

Proyecto de CRUD básico de usuarios con frontend en React (Vite) y backend en Docker (users-api).  
Permite crear, listar, editar y eliminar usuarios, además de iniciar sesión para acceder a las rutas protegidas.
